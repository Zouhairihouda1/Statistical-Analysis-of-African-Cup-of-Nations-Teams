# AFCON Data Analysis

This project performs an exploratory data analysis on African Cup of Nations teams.

## Tools
- Python
- Pandas
- Matplotlib
- Seaborn

## Objectives
- Analyze team performance
- Study correlations between possession, shots and goals
- Visualize results using charts

## Key Insights
- Shots on target have stronger influence than possession
- Efficiency is the most important factor for success
